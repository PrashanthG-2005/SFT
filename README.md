# Smart Finance Tracker App

Full-stack MERN app for expense tracking, budgets, goals, investments.

## Setup
1. Copy `.env.example` to `.env` and update `MONGO_URI` (use MongoDB Atlas/local), `JWT_SECRET`.
2. `npm install`
3. `npm run dev`

Frontend: http://localhost:3000  
Backend: http://localhost:5000

## Features
- Auth (JWT)
- Expenses CRUD
- Budgets, Goals, Investments
- Charts & Reports
- Pure CSS responsive UI, dark mode
